# CRM Logo Assets

This folder contains the CRM integration logos for the Integrations section.

## Required Image Files

Each logo imageset requires the following image files:

### For each logo (fub_logo, kvcore_logo, hubspot_logo, monday_logo, zapier_logo):

**Light Mode:**
- `{logo}_logo.png` (1x - 28x28px recommended)
- `{logo}_logo@2x.png` (2x - 56x56px recommended)
- `{logo}_logo@3x.png` (3x - 84x84px recommended)

**Dark Mode (Optional):**
- `{logo}_logo_dark.png` (1x - 28x28px recommended)
- `{logo}_logo_dark@2x.png` (2x - 56x56px recommended)
- `{logo}_logo_dark@3x.png` (3x - 84x84px recommended)

## Adding Images

1. Download or create the logo images in PNG format
2. Create @2x and @3x versions for Retina displays
3. Optionally create dark mode variants for better visibility
4. Place the files in the corresponding `.imageset` folder
5. The asset catalog will automatically detect and use them

## Notes

- Images should be transparent PNGs with proper alpha channels
- Recommended size: 28x28pt base (28x28px @1x, 56x56px @2x, 84x84px @3x)
- Dark mode variants are optional but recommended for better UX
- If dark mode variants are not provided, the light mode images will be used for both
